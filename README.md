# 📊 SQL Data Analysis Project – Superstore Dataset

## 📌 Project Overview
This project focuses on analyzing retail sales data using SQL to extract meaningful business insights.  
The goal is to understand sales performance, customer behavior, product trends, and profitability.

---

## 📂 Dataset
- Dataset: Superstore Sales Data
- Contains information about orders, customers, products, sales, profit, and regions.

---

## 📁 Project Structure
- dataset/ → contains raw dataset  
- superstore_analysis.sql → contains all SQL queries  
- README.md → project documentation  

---

## 🛠️ Tools & Technologies Used
- MySQL (for data analysis)
- SQL (for querying and data manipulation)

---

## ⚙️ Data Cleaning Steps
- Checked for null values and inconsistencies
- Converted date columns from string to DATE format
- Created new cleaned columns (`updated_order_date`, `updated_ship_date`)
- Verified data accuracy and structure

---

## 📊 Analysis Performed

### 🔹 Basic Analysis
- Total number of orders and sales
- Total profit and average sales per order

### 🔹 Category & Region Analysis
- Sales and profit by category
- Sales distribution across regions

### 🔹 Customer & Product Analysis
- Top customers by purchase value
- Top-selling products

### 🔹 Time-based Analysis
- Monthly and yearly sales trends
- Running total of sales over time

### 🔹 Advanced Analysis
- Ranking products based on sales
- Top 3 products in each category
- Percentage contribution of each category
- Identifying repeat customers

---

## 📊 Key Insights

### 📊 1. Category Performance
- Technology category generates the highest sales and profit.  
- High-value products drive revenue.  
- Focus on expanding Technology product offerings.

### 📉 2. Loss-making Sub-Categories
- Some sub-categories generate negative profit.  
- Likely due to high discounts or poor pricing strategy.  
- Optimize pricing and reduce unnecessary discounts.

### 🌍 3. Regional Sales Distribution
- Certain regions contribute more to overall sales.  
- Indicates stronger demand or better market penetration.  
- Expand operations in high-performing regions.

### 🏙️ 4. Top Cities
- Top 3 cities generate a significant portion of revenue.  
- Driven by higher purchasing power and demand.  
- Focus marketing and campaigns in these cities.

### 👤 5. Customer Behavior
- Some customers make repeat purchases.  
- Indicates strong customer loyalty.  
- Introduce loyalty programs and personalized offers.

### 📦 6. Product Performance
- A small number of products dominate total sales.  
- These are likely popular or essential products.  
- Promote and bundle these products for higher sales.

### 📅 7. Sales Trend Over Time
- Sales vary across months and years.  
- Shows seasonal patterns and business cycles.  
- Plan inventory and marketing based on peak periods.

### 📈 8. Contribution Analysis
- Some categories contribute more to total sales.  
- Reflects product demand and pricing impact.  
- Focus on high-performing categories for growth.

---

## 💡 Business Impact
- Helps identify high-performing products and categories  
- Supports better pricing and discount strategies  
- Enables targeted marketing using customer insights  
- Improves profitability by identifying loss-making areas
